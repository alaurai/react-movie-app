## :construction_worker: How to run

```bash
# Clone repository
$ git clone https://github.com/alauraivani/react-movie-app
   
# Go to project folder
$ cd react-movie-app
   
# Install dependencies
$ yarn install
   
# Run Application
$ yarn start
   
```
